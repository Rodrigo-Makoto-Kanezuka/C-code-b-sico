# C Code Básico - Parte 1

## Sobre o Projeto

Este repositório reúne exercícios e exemplos desenvolvidos durante os estudos iniciais da linguagem C. O objetivo é praticar os conceitos fundamentais da programação estruturada, proporcionando uma base sólida para o desenvolvimento de programas mais avançados.

---

## Objetivos

Aprender a sintaxe básica da linguagem C.

Compreender variáveis e tipos de dados.

Utilizar operadores aritméticos e lógicos.

Desenvolver estruturas de decisão.

Trabalhar com estruturas de repetição.

Criar funções simples.

Praticar lógica de programação.
